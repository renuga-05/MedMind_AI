# Dataset Information

Dataset Name: Heart Failure Clinical Records Dataset

Source: Kaggle

File Type: CSV

Purpose:
This dataset contains patient health information related to heart failure. It helps in understanding patient records and designing a user-friendly Healthcare Patient Portal.

Features:
- Age
- Anaemia
- Creatinine Phosphokinase
- Diabetes
- Ejection Fraction
- High Blood Pressure
- Platelets
- Serum Creatinine
- Serum Sodium
- Sex
- Death Event
